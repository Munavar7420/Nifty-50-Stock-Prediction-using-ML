# Nifty-50-Stock-Prediction-using-ML
## Objective:
#### 1. Analysis of Nifty50's past 4 years data and predicting the future trend for investing.
#### 2. The used algorithm in python will predict the profit or loss based on the data of the the stocks to buy and the return on investment

1.  Scraping four years of NIfty50 Data from Website. 
2.  Analyzed the raw data and extract information from it to identify significant elements that influence stock price using different Python libraries..
4. Analysing & cleaning the data over excel and making the relevant reports over SQL. 
5. The algorithm is for doing the relevant analysis, and building an ML model around it to project future Predictions
6. The Final algorithm will predict the stock price and recommend the client in which company he should invest.
